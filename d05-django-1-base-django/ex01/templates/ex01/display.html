# ...
from django.shortcuts import render
# ...

# ...
def display(request):
    return render(request, 'display.html')
# ...

Explaining the process

  1. Open any browser and search for the link you want to visit (it must be at http://localhost:8000).
  2. The system will check if your request matches any of allowed urls (in the example above it will check firstly in the main project urls and then it will check the app urls).
  3. Then it will return a view that will try to load a .html template.
  4. If the template was found, it will give you a response with the rendered html and the result can be checked in the browser you're using.
  5. If something goes wrong, it will give you an error.

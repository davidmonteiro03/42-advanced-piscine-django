Views

Django views are Python functions that take http requests and return http response, like HTML documents.

A web page that uses Django is full of views with different tasks and missions.

block Template Tag

Definition and Usage

The block tag has two functions:

  1. It is a placeholder for content.
  2. It is content that will replace the placeholder.

In master templates the block tag is a placeholder that will be replaced by a block in a child template with the same name.

Syntax

{% block name %}
  ...
{% endblock %}

Parameters

Value Description
name Specifies that name of the block.

for Template Tag

Definition and Usage

The for tag allows you to iterate over items in an object.

Syntax

{% for item in object %}
...
{% endfor %}

Parameters

Value Description
item Required. Variable name(s) that represents the item(s) of the iterable object.
object Required. An iterable object.

Built-in for Variables

There are some built-in variables you can use inside for loops:

Variable Description
forloop.counter The current iteration, starting at 1.
forloop.counter0 The current iteration, starting at 0.
forloop.first Check if this iteration is the first iteration.
forloop.last Check if this iteration is the last iteration.
forloop.parentloop Refers to the parent loop.
forloop.revcounter The current iteration, counting backwords, ending at 1.
forloop.revcounter0 The current iteration, counting backwords, ending at 0.
